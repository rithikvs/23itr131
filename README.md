# Campus Notification Frontend

A simple React + Vite front-end for placement event notifications and result updates.

## Features

- Demo login with localStorage-based auth
- Dashboard, Events, Results, and Profile views
- Event interest toggles saved in browser storage
- No database required; data is stored locally in the browser

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```
